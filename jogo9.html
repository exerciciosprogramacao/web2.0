📘 Documentação do Projeto - Clicker

🎯 Objetivo do Jogo

  - Clique na imagem para ganhar moedas.
  - Use as moedas para comprar melhorias:
    • Mais moedas por clique
    • Mais moedas por segundo

👉 Um jogo simples onde o jogador evolui clicando e comprando upgrades.

🧱 1. Selecionando os Elementos da Página

  const imagem = document.querySelector('.meio .imagem img');
  const contador = document.querySelector('.contador');
  const contadorSegundos = document.querySelector('.segundos');
  const cliques = document.querySelector('.clique');

👉 Esses comandos pegam os elementos HTML que serão atualizados no jogo.

⚙️ 2. Criando Variáveis Iniciais

  let numeroCliques = 1;
  let numeroSegundos = 0;
  let moedas = 0;

👉 Variáveis para controlar cliques, produção automática e total de moedas.

🔄 3. Atualizando os Valores na Tela

  contador.innerText = "Moedas: " + moedas;
  cliques.innerText = "Cliques: " + numeroCliques;

👉 Exibe as moedas e o valor de cada clique logo ao iniciar o jogo.

👆 4. Clicando na Imagem = Ganha Moedas

  imagem.addEventListener('click', function() {
    moedas += numeroCliques;
    contador.innerText = "Moedas: " + moedas;
    this.style.transform = 'scale(1.1)';
    setTimeout(() => {
      this.style.transform = 'scale(1)';
    }, 20);
  });

👉 Ao clicar, ganha moedas e a imagem faz uma animação de zoom rápido.

🛒 5. Upgrades por Clique

  function botaoc1(){
    if(moedas >= 50){
      moedas -= 50;
      numeroCliques++;
      contador.innerText = "Moedas: " + moedas;
      cliques.innerText = "Cliques: " + numeroCliques;
    }
  }
  // ... igual para botaoc2 até botaoc5

👉 Cada botão aumenta o valor do clique se você tiver moedas suficientes.

⏱️ 6. Upgrades por Segundo

  function botaos1(){
    if(moedas >= 100){
      moedas -= 100;
      numeroSegundos++;
      contador.innerText = "Moedas: " + moedas;
      contadorSegundos.innerText = "Segundos: " + numeroSegundos;
    }
  }
  // ... igual para botaos2 até botaos5

👉 Permite ganhar moedas automaticamente sem precisar clicar.

⏲️ 7. Timer Automático

  setInterval(function() {
    moedas += numeroSegundos;
    contador.innerText = "Moedas: " + moedas;
    contadorSegundos.innerText = "Segundos: " + numeroSegundos;
  }, 1000);

👉 A cada segundo, o jogo adiciona moedas com base nos upgrades automáticos.

🔘 8. Ligando os Botões

  document.getElementById('btc1').onclick = botaoc1;
  // ... btc2 a btc5
  // ... bts1 a bts5

👉 Conecta os botões HTML com as funções de upgrade do JavaScript.
