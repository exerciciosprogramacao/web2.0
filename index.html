Manipulação Simples com JavaScript

🎯 Objetivo

- Manipular o conteúdo de um elemento HTML através de JavaScript.
- Criar um evento de clique no botão para exibir uma mensagem na tela e no console usando uma função.

👉 Este exercício visa entender como usar o JavaScript fora do HTML e interagir com a página.

👉 Repare que temos o <script src="script.js> </script.js>" no fim do nosso body, esse é um arquivo script.js que precisamos criar, ele é como um index.html só que agora para usarmos javaScript.

🧱 1. Selecionando o Elemento de Texto

👉 Vamos selecionar o elemento parágrafo e alterar seu texto.

- Selecionar o parágrafo com id "parag" e ecreva algo que aparece no html

👉 document é usado para acessar o documento html.

👉 getElementById é usado para acessar um único elemento pelo seu id.

🖱️ 2. Criando o Evento de Clique

👉 Temos trê formas de fazer esse evento.

- Somente com evento de onclick no id

👉 Quando o elemento com este ID for clicado, execute a função mensagem". A função mensagem simplesmente exibe uma caixa de diálogo com o texto "Botão clicado!".

- Com addEventListener

👉 var é para dizermos que temos uma variável, var botao, significa que demos o nome de botao a variável.

👉 var botao = o símbolo de igual é para dizermos que nossa variável vai recebe algo.

👉 document.getElementById("but"): Seleciona o botão com o id "but".

👉 .addEventListener("click", mensagem): Adiciona um ouvinte para o evento de "clique" (click) e diz para executar a função mensagem quando isso acontecer.

- Com addEventListener e arrowFunction

👉 cont é uma outra maneira de declarar uma variável, o tipo const é para variavéis que teram sempre o mesmo valor.

👉 const botao = document.getElementById("but") seleciona o botão na página que tem o id "but" e o armazena na variável botao.

👉 botao.addEventListener("click", ...): Adiciona um ouvinte de evento (addEventListener) ao botão. Ele espera por um evento de clique ("click").

👉 () => { ... }); Esta é uma arrow function, uma maneira curta de escrever uma função. O código dentro das chaves ({ ... }) será executado quando o clique ocorrer.
