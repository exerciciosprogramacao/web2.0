Calculadora

🎯 Objetivo do Projeto

- Desenvolver a programação de uma calculadora usando DOM.
- Utilizar a função eval( ) para converter e calcular a operação matemática.

📃 Html

👉 Para esta calculadora, usaremos uma estrutura de lista.

👉 Reproduza a imagem nas tags HTML.

🎮 1. Função e Click

👉 Criamos o evento de clique para cada botão, juntamente com sua respectiva função.

👉 Este processo deve ser feito para todos os botões, 16 no total.

🌎 2. Variáveis globais

👉 Algumas variáveis globais que serão usadas em todo o projeto.

🧮 3. Funções

👉 Dentro da função estamos pegando o valor de texto que temos no HTML e salvando em uma variável.

👉 Em seguida, estamos passando essa variável para o input, que é nossa "tela".

👉 Exemplo: ao apertarmos o botão zero, temos na tag dele o valor "0"; esse valor deve aparecer no nosso input.

👉 Crie uma função para cada botão, apenas deixe de fora a função do "=" e a de apagar "C". São 14 funções no total.

🧮 4. Função de resultado "="

👉 Precisamos pegar todo o valor que está no input (nossa expressão) e passá-lo para uma variável.

👉 Em seguida, usamos a função do JavaScript chamada eval( ); ela serve para interpretar a expressão matemática e identificar qual operador usar primeiro, garantindo que a conta esteja sempre correta.

👉 Por fim, precisamos passar o valor dessa variável — que agora contém a conta calculada — para o nosso input de resultado.

🧮 6. Função de apagar "C"

👉 Para as duas variáveis, passamos o valor de aspas vazias " ".

👉 Por último, só precisamos passar o valor dessas variáveis para os dois inputs, fazendo com que ambos fiquem em branco.
