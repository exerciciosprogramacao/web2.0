Cadastro de Pessoas

🎯 Objetivo do Projeto

- Criar um sistema de cadastro de pessoas, onde informações como nome, e-mail e idade são inseridas em um formulário.
- Cada nova pessoa cadastrada será exibida em um "card" na tela.
- O formulário deve ser limpo após o envio e os dados armazenados em uma lista.

🗃 1. Selecionando os Elementos

👉 Aqui, usamos getElementById para selecionar os elementos do HTML, como o formulário e a área onde os cards serão exibidos. Repare que estamos usando o id na tag form.

📃 2. Declarando a Lista de Pessoas

👉 A variável pessoas é um array vazio que armazenará os objetos das pessoas cadastradas. A cada envio de formulário, um novo objeto será adicionado à lista.

🖱️ 3. Detectando o Envio do Formulário

👉 O addEventListener é usado para escutar o evento de envio do formulário. O evento que queremos capturar é submit.

👉 function(event). Esta é a função de callback que será executada toda vez que o evento de submit ocorrer no formulário. O objeto event é passado para esta função, contendo informações sobre o evento que acabou de acontecer.

👉 Esta é a parte crucial. Quando um formulário HTML é enviado (por exemplo, clicando em um botão de envio), o comportamento padrão do navegador é: a. Recarregar a página. b. Tentar navegar para a URL especificada no atributo action do formulário (ou a mesma página, se não houver um action). c. Anexar os dados do formulário à URL (se o método for GET) ou enviá-los no corpo da requisição (se o método for POST).

👉 👉 O preventDefault evita que a página seja recarregada ao enviar o formulário, algo que acontece por padrão com o evento submit.

✅ 4. Coletando os Dados do Formulário

👉 Usamos getElementById para acessar os valores inseridos pelo usuário nos campos de nome, e-mail e idade.

🧼 6. Criando o Objeto com os Dados

👉 O objeto pessoa é criado para armazenar os dados inseridos pelo usuário no formulário. Isso ajuda a organizar os dados de forma estruturada.

👉 nome, email e idade são chamados de propriedades. Neste caso propriedades do objeto pessoa.

👁️ 7. Adicionando o Objeto à Lista, chamando a função e resetando formulário

👉 A função javaScript "push" adiciona o objeto [🙍‍♂️] à lista de pessoas [📑], armazenando todos os cadastros realizados.

👉 exibirCards() é uma função, ainda não criamos ela, ela vai servir para mostrar os cards.

👉 📮.reset() é uma função javaScript para resetar os campos do formulários

🔍 8. Função de Exibir os Cards

function exibirCards()
[📦].innerHTML = '';

// Para cada pessoa, criamos um card
[📑].forEach([📚] => {
  const [🎴] = document.createElement('div');
  [🎴].classList.add('[......]');
  [🎴].innerHTML = 
        "<h3>" + [📚].nome + "</h3>" +
        "<p>Email: " + [📚].email + "</p>" +
        "<p>Idade: " + [📚].idade + "</p>";

  // Adicionando o card à área de exibição
  [📦].appendChild([🎴]);
});

👉 Aqui, usamos forEach para percorrer a lista de pessoas [📑] e criar um card para cada item da lista. Cada card é adicionado ao container de cards [📦].
