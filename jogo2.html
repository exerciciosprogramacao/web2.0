📘 Exercício de createElement - Cadastro de Pessoas

🎯 Objetivo do Projeto

- Criar um sistema de CRUD (Criação, Leitura, Atualização e Deleção) simples.
- Permitir ao usuário adicionar novas linhas (pessoas) em uma tabela.
- Permitir ao usuário remover linhas existentes com base nos dados informados.

👉 Este exercício foca em criar e manipular elementos HTML (especificamente linhas e células de tabela) diretamente no DOM (Document Object Model) usando JavaScript puro.

🖱️ 1. Atribuindo Eventos aos Botões

👉 O método document.getElementById( ) busca um elemento HTML pelo seu atributo id. Usamos ele para localizar os botões de adicionar e remover.

👉 Ao atribuir uma função como adicionarPessoa à propriedade .onclick do elemento, dizemos ao navegador para executar essa função sempre que o botão for clicado.

➕ 2. Função adicionarPessoa( ) - Criando Elementos

👉 Primeiro, pegamos os valores que o usuário digitou nos campos de input usando .value.

👉 Em seguida, document.createElement( ) é usado para criar novos elementos HTML na memória: uma linha de tabela <tr> e duas células de dados <td>.

👉 A propriedade .innerText é usada para inserir o texto (nome e idade) dentro das novas células.

📤 3. Função adicionarPessoa( ) - Inserindo na Tabela

👉 O método .appendChild( ) conecta os elementos, criando a hierarquia: as células são adicionadas à nova linha novaLinha, e a linha completa é adicionada à tabela identificada por tablepeople.

👉 No final, os campos de input são limpos, reatribuindo uma string vazia (" ") ao seu valor .value.

⛪ 4. Função removerPessoa( ) - Buscando e Comparando

👉 Esta função é a mais complexa, pois ela itera "percorre" todas as linhas da tabela para encontrar e remover a entrada que corresponde ao Nome e Idade digitados pelo usuário.

👉 Primeiro, pegamos os valores que o usuário digitou nos campos de input usando .value.

👉 document.getElementsByTagName"tr" obtém todas as linhas <tr> da página.

⛪ 4.1. Função removerPessoa( ) - for( ){

👉 O loop for começa em i = 1 para ignorar a primeira linha, que geralmente é o cabeçalho th e não deve ser analisado.

👉 Dentro do loop, pegamos o texto exato da primeira célula [0], que é o nome cadastrado e da segunda célula [1], que é a idade cadastrada. Isto para cada linha. Começando da linha 1 pois a 0 é a th.

⛪ 4.2. Função removerPessoa( ) - if( ){

👉 A condição if verifica se o nome e a idade digitados no input são idênticos aos valores das variáveis que receberam o que estava na td da primeira linha, que armazena o nome e na outra td desta mesma linha, que armazena a idade.

👉 Se a condição if for verdadeira, o método tabela.deleteRow(i) é chamado. Ele usa o índice i para remover a linha da tabela permanentemente. O return encerra a função imediatamente após a remoção.

👉 O else serve quando o nome e a idade digitados não são iguais ao armazendo na tabela, isso quer dizer que não podemos deletar, pois não achamos ninguém.

⛪ 4.3. Função removerPessoa( ) - Limpando inputs

👉 No final, os campos de input são limpos, reatribuindo uma string vazia (" ") ao seu valor .value.
