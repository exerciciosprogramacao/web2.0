Alternância de tema

🎯 Objetivo do Projeto

- Alternar entre dois estados visuais (temas), mudando as classes CSS aplicadas ao elemento body.

📃 1. HTML

👉 HTML básico com todas as classes que usaremos.

👾 2. JavaScript

👉 "Ativamos" a função por meio do mapeamento do ID.

👉 O método classList.toggle() verifica se a classe "modewhite" já existe no elemento body. Se existir, a classe é removida; caso contrário, é adicionada.

🖼 3. CSS

👉 À esquerda temos o CSS padrão, com tamanhos, espaçamentos etc. À direita, temos o CSS responsável apenas pelas mudanças de cores e pela aplicação da classe toggle no elemento pai body. Todo o CSS do nosso tema terá o formato body.modewhite (sem espaços). Depois, adicionamos a tag ou classe, como em body.modewhite h2.

👉 O "truque" na mudança de cor, na verdade, está no CSS, que contém todas as classes com os dois modos. O JavaScript apenas faz a troca entre elas.

⛪ 4. Desafio

👉 Escolha três de seus projetos e implemente essa troca de tema, seja por um botão ou da forma que preferir.

💰 Este desafio vale 20 moedas pelos três projetos.
