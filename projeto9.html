Calculadora Refatorada

🎯 Objetivo do Projeto

- Refatorar a programação da calculadora, ou seja, melhorar o código, diminuindo a quantidade de linhas e ganhar desempenho.
- Utilizar querySelectorAll, forEach e addEventListener.

📃 Dicas

👉 Precisamos que todos os botões menos o botão de apagar e o de igual tenham a mesma classe. Cuidado com o CSS — uma solução é usar ID no CSS com o seletor #.

👉 Você já aprendeu tudo o que precisa...

👉 Precisamos usar o querySelectorAll para pegar todos os botões com a classe escolhida e passar essa NodeList para uma variável.

👉 Nessa variável usamos um forEach com arrow function e um parâmetro.

👉 Dentro do forEach — ou seja, para cada botão — teremos um addEventListener que escutará o clique do mouse.

👉 Dentro da função que utiliza o clique, temos que usar o parâmetro para passar para o display o valor dos botões clicados, usando innerText.

👉 Fora dessas duas arrow functions, teremos apenas o mapeamento dos IDs de apagar e de igual, e também uma função para cada um desses dois botões. Você pode copiar esta parte do projeto antigo — é a mesma programação.

👉 Como resultado, teremos, contando as linhas em branco, cerca de 24 linhas de código.
